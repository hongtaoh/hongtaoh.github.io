ChatGPT 都出来 3 年了，我作为一个计算机博士生，直到现在才了解大语言模型背后的原理。起因是我在找工作，那个公司要求做一个后训练 (Post Training, 具体来说是 RLHF) 的系统。我一开始觉得很难，因为我从来没有接触过。截止日期当天，我也没事做，就说试试吧，结果[不到三个小时就弄完了](https://github.com/hongtaoh/TriFetch_Code)。背后的数学原理我觉得和初中数学，甚至小学数学差不多。这让我开始反思，连我一个 CS 在读博士都被那些看起来很高端的术语比如 Post-training, SFT, RLHF, DPO, LoRa, GRPO 给唬住了，我很难想象一个大学文科毕业的学生会有一丝一毫自己想去学的冲动，以及觉得自己可以学会的信心。

过了第一轮，这家公司说要进入技术面试，要考察我对 Transformer 架构的理解。我说实话并不理解。前几天我说试一下看看，我都没去看 Attention is all you need 的论文，我就只是说我从零开始自己写底层代码训练一个 BERT 吧。结果我用了大概两天就差不多弄完了。我不敢说我精通了，但是我可以自己从零开始训练 BERT 了。懂了 BERT，然后也就很轻松地理解了大语言模型那个 Predict next token 到底是怎么回事，以及如何实现，这部分的底层代码我还在写。

我之前买了 Build a Large Language Model From Scrath 这本书，但是看了前两章这本书就开始吃灰了，实在没动力跟着他做下去。我自己从零开始写 BERT 之后，我觉得自己动手写比看教材要高效很多。你跟着别人的思路走一遍，和自己从零开始逼着自己把路走通，那种感觉是完全不同的。

我说实话，我觉得让一个文科毕业的大学生入门大语言模型只需要三篇博客文章就可以了。我说的入门是说可以从底层自己写代码，自己训练，而不是那种泛泛的理解。第一篇讲一下 Python 的基础和需要用到的数学基础（非常少，初中数学就可以）。第二篇写 BERT。第三篇写 GPT。如果非要加一篇，那第四篇写后训练，但这是延伸。

但是，三篇博客怎么赚钱？对吧。而且你把这些东西弄得这么容易理解，不是显得自己的专业很没有门槛吗？所以呢，很多书就出来了，很厚很厚的书。毕竟书厚实了，作者觉得自己很牛，读者也觉得东西很厚重也愿意付钱，皆大欢喜，不是吗？注意，我在这里并不是说 Build a Large Language Model From Scrath 这本书。这本书写得非常好！我只是在说一个普遍的现象。

或者出非常多的视频。拿爬虫来说，我真的觉得我用[两篇博客](https://github.com/hongtaoh/webscraping-tutorial) 就把爬虫的基础全都覆盖了，但是并不妨碍有那些非常多的视频教程。

我现在觉得在这个信息时代，买方和读者真的需要转换一下心态。真正值钱的不是很厚的东西，而是用最短时间能把问题讲清楚的内容。

现在很多媒体在文章里的共识是，大家要学一下 AI。但是 AI 到底学什么？普通人怎么学？并没有人回答。

甚至不是「普通人」。我认识理工科毕业的博士，都不知道 AI 怎么学、学什么。现在是缺教程吗？并不是啊，网上一搜一大把。在贩卖的课程也是一大把。

我在[之前的博客](/cn/2024/12/22/baiwan/)中提到过，现在的 AI 教程，大部分是贩卖焦虑、不懂装懂、漏洞百出、只为敛财的垃圾课程，要么是对大众并不友好的教程，中间态基本没有，至少我没看到过。为什么？原因有很多。第一，懂了的人，很多都很聪明，很难理解那些基础的东西为什么需要解释。比如，你要讲 BERT，一个连矩阵相乘都不知道的人，怎么可能理解你的东西。第二，把 AI 讲清楚需要花很多时间，而且收益是不确定的。有这个时间，还不如好好工作赚钱。第三，我猜，懂了的人并不觉得有义务把这些知识告诉所有人。

我举一个例子。我昨天看到了 [minimind](https://github.com/jingyaogong/minimind) 这个项目，甚至连[多模态](https://github.com/jingyaogong/minimind-v)的都出来了。他这个是实打实的从零开始自己写。但是，代码写出来了，并没有人去讲背后的原理。我不相信一个大学毕业的文科生，只通过这个项目，就学会了大语言模型。写这些仓库可能需要几天，但是要想把这些原理给普通人讲清楚，我觉得需要几周甚至几个月。很少人会想去这么做。「我现在工作都没有，对我来说最重要的是自己生存下去，并不是去教会别人。」我猜财务自由之前的人大部分应该是这种心态，至少我是这么想的。确实没必要耗费那样的精力。

有解法吗？我不知道。

那现在回答一下标题的问题。普通人如何学会 AI？

我觉得要有一个具体的目标。比如，我觉得一个很合适的目标是，自己从零开始把 BERT 训练出来，可以用很小的训练数据。注意力层限制在 2 层。让 AI 一步步教你，你也要确保每一步都自己彻底理解。不要去看教材和目前的教程，就自己从零开始写底层代码。

为了避免大家觉得我在「口嗨」，我用我自己之前写过的教程来给一个具体的路径。我假设你是一个初中毕业的学生，高中都没上。数学一窍不通，更没有接触过编程。

- 先去买一台破电脑。2000 人民币以内。
- 我没有写过这种安装软件的教程，但是我可以说一下找什么：
  - 下载并安装 python
  - 下载并安装 VS Code
- 问 DeepSeek，让它帮你 30 分钟之内入门 Python + Jupter Notebook + VS Code。
- 跟着[我之前的教程](https://hongtaoh.com/cn/2024/03/22/personal-website-tutorial/) 自己做一个个人网站。
- 把我写的[逻辑回归](https://hongtaoh.com/cn/2023/04/24/lr/) 彻底理解，自己写一下代码。中间有不同的数学，自己用 AI 理解一下。
- 把我写的[神经网络](/cn/2023/04/26/nn/) 彻底理解，自己写一下代码。中间有不同的数学，自己用 AI 理解一下。
- 把我讲 [RAG](https://hongtaoh.com/cn/2025/03/16/bert/) 的这篇彻底理解，并自己写代码。

然后，你就可以跟着 AI 一起从零开始训练 BERT。
